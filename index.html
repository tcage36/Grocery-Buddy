IPHONE SHORTCUT

Grocery to Kroger

Build it once, then launch it directly from Grocery Buddy.

Shortcut actions

  1. Create a new shortcut named exactly Grocery to Kroger.
  2. Add Find Reminders. Filter for your Grocery list and Is Completed is false. Sort oldest first.
  3. Add Repeat with Each using the reminders found.
  4. Inside Repeat, get the reminder’s title, then Split Text using the custom separator — .
  5. Add Get Item from List → First Item, then Copy to Clipboard.
  6. Add Show Alert: “Copied [First Item]. Paste it into Kroger search, add the product, then return to Shortcuts.”
  7. Add Open App → Kroger.
  8. Immediately after Open App, add Wait to Return. This keeps the shortcut on the current item while you paste, choose a product, and add it in Kroger.
  9. After you return, add a menu to mark the current reminder complete, skip it, or stop. Then Repeat moves to the next reminder.

Apple sometimes changes the exact wording of an action, but the action names above are searchable in the Shortcuts editor. A printable setup guide is also included in the app folder.
