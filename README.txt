GROCERY BUDDY FOR IPHONE — VERSION 1.4.0

BASELINE
- Built directly from the tested Version 1.3.1 app.
- Preserves meal planning, per-day styles, Quick filters, swaps, side choices, checked-item hiding, Reminders sharing, and saved-plan migration.

WHAT CHANGED
- Replaced generic Google recipe searches with exact in-app recipes and prep instructions.
- Every meal now stores its own ingredient quantities, cooking steps, serving baseline, and source record.
- Grocery quantities are now calculated from the exact selected recipes instead of one generic ingredient table.
- Package items clearly distinguish what to buy from how much the recipes actually use (for example: buy 1 can; recipes use 3/4 can).
- Expanded every meal category to 8 curated choices: 48 total meals.
- Selected side dishes now also have exact quantities and prep instructions and are included in the recipe view and grocery list.
- Fixed the Grocery Buddy icon face so the eyes are above the smile.
- Improved update caching so a newly uploaded version is less likely to remain stuck behind the old Home Screen cache.

KROGER NOTE
Version 1.4.0 continues to produce a clean quantity-based list for manual entry into the Kroger app. Direct Kroger account, store-price, coupon, and cart integration requires supported API access and secure account authorization and is not included in this offline web-app build.

INSTALL / UPDATE
Upload this folder’s contents to the same web host used for Version 1.3.1. Open the hosted page in Safari and refresh once before reopening the Home Screen app. If iPhone still keeps the old icon or version, remove the old Home Screen icon and add the refreshed page again.
